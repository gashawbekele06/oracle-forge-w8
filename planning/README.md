## AI-DLC Inception Structure

Every Inception document must contain three sections:

1. **Press Release** — Written in present tense as if the sprint is complete. Forces the team to agree on what is being built before building it.
2. **Honest FAQ** — What could go wrong? Two perspectives: user-facing and technical.
3. **Definition of Done** — Numbered, verifiable checklist. Each item is a bash command or manual check with expected output.
